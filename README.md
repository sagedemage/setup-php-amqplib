# Setup php-amqplib

## Dependencies
Install system dependencies
```
sudo apt install php php-amqp php-amqplib composer
```

Install project dependencies
```
composer install
```

## Run the program
Run the receiver
```
./receive.php
```

Run the sender
```
./send.php
```
